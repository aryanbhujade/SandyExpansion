03 · AI-powered talent discovery

From a question to a grounded recommendation

User Query"Who knows about Kubernetes?"
➡
Request AnalyserOllama parses intent, topic & skills
➡
Context BuilderSQLite token-overlap match on responsibility topics
➡
Recommendation Enginehierarchy-aware scoring
➡
Answer GeneratorOllama synthesizes the response
➡
Recommendationsranked experts + reasoning

How matching works

  • Natural-language parsing — LLM extracts intent, topic & required skills
  • SQLite token-overlap match — queries responsibility topics, maps primary / backup contacts

Hierarchy-aware scoring

  • Ranks experts by department, seniority-level matching & feedback loops
  • Escalation boundaries avoid overwhelming executives
  • Intelligent response — Ollama presents recommendations with logical explanations

08 · Roadmap & gaps to production

From prototype to enterprise-wide

Implemented core

  • AI talent discovery pipeline (Ollama + SQLite match)
  • Hierarchy-aware ranking & grounded answers
  • Peer-to-peer messaging with unread badges
  • Floating toasts & notification logging
  • Self-service profiles with live sync
  • Auth (JWT + bcrypt) & org hierarchy tree

Next steps / production migration

  • Seed full scale — seed SQLite with the full 450-employee pool (generator exists, needs DB integration)
  • Vector database migration — move keyword/token search to ChromaDB for true semantic-vector search
  • Enterprise DB integration — migrate SQLite to PostgreSQL for durability & concurrency
  • Teams & Sandy Bot integration — ship the standalone prototype into Microsoft Teams as an official corporate bot
